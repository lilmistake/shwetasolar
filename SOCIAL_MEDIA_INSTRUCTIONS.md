# How to Add Social Media Posts

## Quick Start

1. **Prepare Your Post Information:**
   - Post title (optional, but recommended)
   - Post content/caption
   - Image (if applicable)
   - Link to original post on social media
   - Date published

2. **Share With Me:**
   Simply provide the posts in this format:

   \`\`\`
   POST 1:
   - Title: [Your post title]
   - Content: [Your post text/caption]
   - Image: [Attach image or describe it]
   - Platform: [LinkedIn/Twitter/Facebook/Instagram/All]
   - Link: [URL to original post]
   - Date: [When it was posted]

   POST 2:
   ...
   \`\`\`

3. **I'll Create the SQL Script:**
   I'll generate a SQL script to add all your posts to the database.

4. **Run the Script:**
   Execute the script in your Supabase SQL editor (I can do this for you).

## Tips

- **Use "All" as platform** since your content is the same across all platforms
- **Include high-quality images** for better engagement
- **Keep content concise** - 2-3 sentences work best
- **Add links** to drive traffic to your social media profiles
- **Update regularly** - Add new posts as you publish them

## Bulk Upload

If you have many posts, you can:
1. Share them all at once in a document
2. Provide a spreadsheet with columns: Title, Content, Image, Link, Date
3. I'll create a single SQL script to add them all
